# Backend Codemap

> Freshness: 2026-02-01

## Status: No Backend

Emot-ID is a fully client-side PWA. There is no backend, API, or database.

## External Services

| Service | Usage | Integration |
|---------|-------|-------------|
| Google AI Search | Emotion exploration | URL link with `udm=50` param (AI mode) |
| GitHub Pages | Static hosting | Vite build output at `/emot-id/` base path |

## Data Storage

| Storage | Mechanism | Data |
|---------|-----------|------|
| Language preference | localStorage (`emot-id-language`) | 'ro' \| 'en' |
| Emotion data | Bundled JSON | Static, no fetching |

## PWA Configuration

- Service worker generated by `vite-plugin-pwa`
- Workbox for precaching (8 entries)
- Manifest at `dist/manifest.webmanifest`
- Offline-capable (all assets precached)

## Build & Deploy

```
npm run build → tsc -b && vite build → dist/
Deploy target: GitHub Pages (static)
Base path: /emot-id/
```
