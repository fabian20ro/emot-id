Interactive app for identifying and exploring emotions, based on Plutchik's Wheel of Emotions.

## What does it do?

- **Select emotions** - Tap the bubbles that resonate with your current state
- **Discover combinations** - The app automatically detects dyads (combinations of 2 primary emotions)
- **Explore with AI** - Direct link to Google AI for deep analysis

## Features

- Modern design with smooth animations (Framer Motion)
- Bilingual support (Romanian / English)
- Responsive - works on mobile and desktop
- Sound feedback on selection

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```
